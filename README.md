```sh
CLIENT_ID=<YOUR_CLIENT_ID>
CLIENT_SECRET=<YOUR_CLIENT_SECRET>
```
Generate your credentials: https://profile.intra.42.fr/oauth/applications/new


```sh
export $(cat env.template | xargs)
```
